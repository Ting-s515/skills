# 功能流程：使用者管理

## 功能概述

管理員可在後台查看、搜尋使用者列表，查看個別使用者詳情，並執行停用或啟用帳號操作。

## 流程說明

### 1. 查看使用者列表

1. 管理員進入「使用者管理」頁面
2. 系統載入使用者列表（預設顯示全部，依建立時間降序排列）
3. 列表顯示欄位：姓名、email、建立時間、帳號狀態

### 2. 搜尋使用者

1. 管理員在搜尋欄輸入 email 關鍵字
2. 系統即時過濾列表，顯示 email 包含該關鍵字的使用者
3. IF 無符合結果 → 顯示「查無使用者」提示
4. IF 清空搜尋欄 → 恢復顯示全部使用者

### 3. 查看使用者詳情

1. 管理員點擊列表中某筆使用者
2. 系統跳轉至詳情頁面
3. 顯示資訊：姓名、email、建立時間、訂單數量、帳號狀態

### 4. 停用 / 啟用帳號

1. 管理員在列表或詳情頁點擊「停用」或「啟用」按鈕
2. 系統顯示確認對話框
   - IF 管理員確認 → 送出 API 請求更新帳號狀態
     - IF API 成功 → 更新 UI 顯示新狀態，顯示成功訊息
     - IF API 失敗 → 顯示錯誤訊息，狀態不變
   - IF 管理員取消 → 關閉對話框，不做任何變更
