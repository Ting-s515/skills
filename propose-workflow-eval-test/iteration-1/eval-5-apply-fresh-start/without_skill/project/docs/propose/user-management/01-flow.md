# 功能流程：使用者管理

## 功能概述

管理員可在後台查看使用者列表、查看使用者詳情，並停用/啟用帳號。

## 流程說明

### 1. 使用者列表

1.1 管理員進入 /admin/users 頁面
1.2 系統呼叫 GET /api/admin/users 取得使用者清單
1.3 列表顯示：姓名、email、狀態、建立時間
1.4 管理員可在搜尋框輸入 email 關鍵字進行篩選
  - 有輸入：呼叫 GET /api/admin/users?email=<keyword>
  - 無輸入：顯示全部

### 2. 使用者詳情

2.1 管理員點擊列表中某使用者
2.2 系統呼叫 GET /api/admin/users/:id 取得詳情
2.3 顯示：姓名、email、建立時間、訂單數量

### 3. 停用/啟用帳號

3.1 管理員在詳情頁點擊「停用」或「啟用」按鈕
3.2 系統呼叫 PATCH /api/admin/users/:id/status
3.3 成功後更新畫面狀態
3.4 失敗時顯示錯誤訊息
