# 功能流程：使用者管理

## 功能概述

管理員可在後台查看、搜尋使用者列表，查看使用者詳情，並對帳號進行停用或啟用操作。

## 流程說明

### 主流程：查看使用者列表

1. 管理員進入使用者管理頁面
2. 系統載入所有使用者列表（分頁顯示）
3. 管理員可輸入 email 關鍵字進行搜尋
   - 有結果：顯示符合條件的使用者列表
   - 無結果：顯示「查無資料」提示

### 子流程：查看使用者詳情

1. 管理員點擊列表中的某筆使用者
2. 系統顯示詳情頁，包含：
   - 姓名
   - Email
   - 建立時間
   - 訂單數量

### 子流程：停用/啟用帳號

1. 管理員在詳情頁或列表中點擊「停用」或「啟用」按鈕
2. 系統顯示確認對話框
3. 管理員確認操作
   - 確認：系統更新帳號狀態，顯示成功訊息
   - 取消：關閉對話框，不做任何變更
4. 操作成功後，列表中的帳號狀態即時更新

### 邊界條件

- 搜尋 email 為空時，顯示全部使用者
- 停用後的使用者無法登入系統
- 已停用的帳號顯示「啟用」按鈕；已啟用的帳號顯示「停用」按鈕
