# 功能流程：報表匯出

## 功能概述
管理者可依日期區間與商品類別篩選，將銷售報表匯出為 Excel 或 CSV 格式。

## 流程說明

### 比對條件
- `start_date`、`end_date`（必填）
- `category_id`（可選，NULL 代表全部類別）
- `format`：`excel` 或 `csv`

### 處理流程

1. 管理者填寫篩選條件並送出匯出請求
2. 輸入驗證：
   - **`start_date` 或 `end_date` 為空：** 回傳 400，結束流程
   - **`start_date` 晚於 `end_date`：** 回傳 400
   - **日期區間超過 365 天：** 回傳 400
   - **`format` 不為 `excel` 或 `csv`：** 回傳 400
3. 查詢已完成訂單（`status = 'completed'`），依日期區間及 `category_id` 篩選
4. **查詢結果為空：** 產生僅含欄位標題的空報表
   **查詢結果不為空：** 產生含資料的報表
5. 依 `format` 產生檔案（excel → `.xlsx`，csv → `.csv` UTF-8 with BOM）
6. 以 HTTP 附件形式回傳供下載

### 邊界條件
- 未登入：401；無匯出權限：403
- 查詢結果為空：回傳僅含標題的空報表，不回傳 404
