# 後台管理系統規格

## 商品管理

管理者可以在後台新增和編輯商品資訊。商品欄位包含：名稱（必填）、售價（必填，需大於 0）、庫存數量（必填，需大於等於 0）。
新增時若驗證失敗，顯示對應錯誤訊息。儲存成功後商品列表即時更新。

> propose: `backend/docs/propose/product-management/`

## 訂單管理

管理者可以查看所有訂單列表，支援依訂單狀態篩選（待付款 / 已付款 / 已出貨 / 已完成）。
每筆訂單顯示：訂單編號、建立時間、客戶名稱、總金額、目前狀態。

> propose: `backend/docs/propose/order-management/`
